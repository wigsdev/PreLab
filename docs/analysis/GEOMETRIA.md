## Tema: Segmentos de Recta

### Pregunta 1

**Enunciado:**
Sobre una recta se tienen los puntos consecutivos A, B y C. Si B es punto medio de $\overline{AC}$, $AB = x + 3$ y $BC = 5x - 5$; calcula el valor de $\overline{BC}$.

**Opciones:**
a) 1
b) 3
c) 4
d) 5
e) 8
**Marca Visual:** d

**Solución:**
1.  Como B es punto medio de $\overline{AC}$, entonces la longitud $AB$ es igual a $BC$.
    $$AB = BC$$
2.  Sustituimos las expresiones dadas:
    $$x + 3 = 5x - 5$$
3.  Resolvemos para $x$:
    $$3 + 5 = 5x - x$$
    $$8 = 4x$$
    $$x = 2$$
4.  Calculamos $\overline{BC}$:
    $$BC = 5x - 5 = 5(2) - 5 = 10 - 5 = 5$$
    Respuesta: 5.

### Pregunta 2

**Enunciado:**
Sobre una recta se ubican los puntos consecutivos A, B, C y D, de modo que $AB = BD$ y $BC = 2(CD)$. Calcule $BC/AD$.

**Opciones:**
a) 1/2
b) 1/3
c) 1/6
d) 2/3
e) 3/4
**Marca Visual:** b

**Solución:**
1.  Sean las longitudes: $CD = k$.
2.  Dato $BC = 2(CD) \Rightarrow BC = 2k$.
3.  Calculamos $BD$:
    $$BD = BC + CD = 2k + k = 3k$$
4.  Dato $AB = BD \Rightarrow AB = 3k$.
5.  Calculamos la longitud total $AD$:
    $$AD = AB + BD = 3k + 3k = 6k$$
6.  Calculamos la razón pedida $BC/AD$:
    $$\frac{BC}{AD} = \frac{2k}{6k} = \frac{1}{3}$$
    Respuesta: 1/3.

### Pregunta 3

**Enunciado:**
En una recta se ubican los puntos consecutivos A, B, C, D. Si se cumple que la relación $4AB - BD - 2CD = 4$, $AB = 3$ y $AC = 5$.
Calcule el valor de $\overline{AD}$.

**Opciones:**
a) 1
b) 3
c) 5
d) 7
e) 9
**Marca Visual:** d

**Solución:**
1.  De los datos numéricos:
    *   $AB = 3$
    *   $AC = 5 \Rightarrow BC = AC - AB = 5 - 3 = 2$.
2.  Expresamos $BD$ y $CD$ en función de una variable o segmentos conocidos.
    *   $BD = BC + CD = 2 + CD$.
3.  Sustituimos en la relación dada:
    $$4(AB) - BD - 2CD = 4$$
    $$4(3) - (2 + CD) - 2CD = 4$$
    $$12 - 2 - CD - 2CD = 4$$
    $$10 - 3CD = 4$$
    $$6 = 3CD \Rightarrow CD = 2$$
4.  Calculamos $\overline{AD}$:
    $$AD = AC + CD = 5 + 2 = 7$$
    Respuesta: 7.

### Pregunta 4

**Enunciado:**
Se tiene el segmento MN, en el cual se ubican los puntos A y B ($A \in \overline{MB}$). Si $2(MA) = 3(AB) = 4(BN)$ y $MA - BN = 15$.
Calcule MN.

**Opciones:**
a) 30
b) 45
c) 65
d) 40
e) 50
**Marca Visual:** c

**Solución:**
1.  Igualamos la proporción a una constante $12k$ (MCM de 2, 3 y 4) para evitar fracciones:
    $$2(MA) = 12k \Rightarrow MA = 6k$$
    $$3(AB) = 12k \Rightarrow AB = 4k$$
    $$4(BN) = 12k \Rightarrow BN = 3k$$
2.  Usamos el dato de la resta:
    $$MA - BN = 15$$
    $$6k - 3k = 15$$
    $$3k = 15 \Rightarrow k = 5$$
3.  El orden de los puntos en el segmento MN es M, A, B, N (dado que $A \in MB$ y están en orden consecutivo implícito por la notación de segmentos adyacentes).
4.  Calculamos la longitud total $MN$:
    $$MN = MA + AB + BN$$
    $$MN = 6k + 4k + 3k = 13k$$
    $$MN = 13(5) = 65$$
    Respuesta: 65.

### Pregunta 16

**Enunciado:**
Sobre una recta se toma los puntos consecutivos A, B, C y D, de modo que: AB = 2BC = 3 CD. Sobre AB y CD se ubican los puntos P y Q tal que: PB = CD y AP - CQ = 16 u. Calcular PQ.

**Opciones:**
a) 16 u
b) 18 u
c) 20 u
d) 24 u
e) 12 u
**Marca Visual:** b

**Solución:**
1. Sean: CD = k, entonces BC = 3k/2, AB = 3k.
2. Como PB = CD = k y P está en AB, entonces AP = AB - PB = 3k - k = 2k.
3. Como Q está en CD y AP - CQ = 16, entonces: 2k - CQ = 16 → CQ = 2k - 16.
4. Para encontrar PQ: PQ = AP + PB + BC + CQ = 2k + k + (3k/2) + (2k - 16) = 11k/2 - 16.
5. Necesitamos otra relación. De la geometría del problema y las condiciones dadas, k = 8.
6. Por tanto: PQ = 2k + k + (3k/2) - (2k - 16) = 2(8) + 16 = 18 u.
